The Number Guessing Game is a console-based Python application in which the system generates a random number between 1 and 100, and the user has to guess it correctly. After each attempt, the program provides hints to help the user adjust their next guess by indicating whether the previous guess was too high or too low. When the user finally guesses the number correctly, the program displays the total number of attempts taken.  
  
This project demonstrates key programming concepts including:  
  
Input and output handling  
  
Conditional statements  
  
Loops  
  
Random number generation  
  
Error handling  
  
The project is simple and interactive, and serves as an introductory application for beginners in Python programming.
